Galaxy Strike
UNITY C Sharp
Galaxy Strike is a thrilling 3D space shooter game where, to protect the galaxy, player has to navigate through enemy territories, destroy hostile enemy ships, dodge obstacles and reach to a safe place.
Programming Features
  • Unity Timeline: Utilized Unity Timeline to create a predefined flight path for the spaceship.
  • Movement Mechanics: Designed smooth and dynamic player movement using Quaternion.Euler() for rotation and Mathf.Clamp() for restricting movement.
  • Unity Terrain Tools: Utilized Unity Terrain to design a realistic and interactive game environment.
  • Score System: Implemented a responsive scoring mechanism that updates in real-time during gameplay.
  • Particle system: Created visually engaging bullet trails and explosion effects to enhance gameplay visuals.
  • Skybox, Lighting & Post Processing: Enhanced visual aesthetics with custom skyboxes, dynamic lighting setups, and advanced post-processing effects.
