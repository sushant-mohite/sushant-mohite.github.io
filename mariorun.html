Mario Run
UNITY C Sharp
A 3D platformer obstacle game inspired by classic platformer games. Player navigate a challenging environment filled with obstacles, traps, and collectibles, aiming to complete the level in the fastest time possible.
Programming Features
  • Programming Patterns: Implemented Singleton Pattern, State Machine, OOP Principles, Scene Management for cleaner and more efficient code.
  • System Design: Developed systems such as the Health / Life System, Checkpoint System and Player Respawn Mechanism.
  • Animations: Implemented smooth Idle, Walk, and Run animations, along with dynamic object animations.
  • UI Management: Built an intuitive Start Menu, Player Lifes UI, Game Win Screen and Game Over Screen for a polished user interface.
  • Physics Collision: Integrated Rigidbody Physics, Triggers, and Collision Detection for interaction and obstacle behavior.
  • Performance Optimization: Used Occlusion Culling to optimize rendering performance, Enabling and Disabling Objects dynamically for memory efficiency, Marked Static GameObjects to reduce memory overhead and improved frame rates.
